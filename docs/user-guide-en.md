2. Click **"Connect Wallet"** button
3. Select MetaMask wallet
4. Approve the signature request for authentication
5. A success message will appear

## System Administration (Admins Only)

### Add User
1. Go to **Admin**
2. Click **+ Add User**
3. Enter wallet address and name
4. Select appropriate role
5. Click **Add**

### Change Permissions
1. Find the user
2. Click on current role
3. Select new role

### Security Settings
- Two-factor authentication
- Multi-signature approval
- Transaction limits
